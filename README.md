# Personal Webpage

A personal webpage built from scratch using HTML and CSS.

## What it is

A personal profile page featuring:
- Name and profile photo
- About Me section
- Interests displayed as styled tags
- Favourite Websites section
- Dark/Light mode toggle

## What I used

- HTML5
- CSS3 (Flexbox)
- JavaScript (DOM manipulation)

## What I learned

- Building a page layout from scratch without templates
- Styling with a consistent color scheme
- Using JavaScript to toggle classes and switch between dark and light mode
- Connecting a JS file to HTML using the `<script>` tag

## How to run it

1. Clone the repo:
   ```bash
   git clone https://github.com/17adarshdash-commits/personal-webpage.git
   ```
2. Open `index.html` in your browser
